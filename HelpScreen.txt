This is the help.
The syntax for commands is
command <parameter> <parameter>

Note that some commands do not use parameters

List of commands:

f <x> <y> 	- toggles the flag at the given coordinate
c <x> <y>	- clicks at the given coordinate
h - prints the help screen
p - prints the discovered map ( happens after every click )
sp - prints the whole map ( use for debugging )
